# Sweta Singh Portfolio

Premium React + Vite portfolio for an Electrical and Electronics Engineering undergraduate focused on Embedded Systems, IoT, Robotics, and Automation.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Stack

- React + Vite
- Tailwind CSS
- Framer Motion
- Lucide React Icons
- CSS variable theming with persisted light/dark mode
